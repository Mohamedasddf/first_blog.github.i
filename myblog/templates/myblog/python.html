دورة python المجانية للمبتدئين

انطلق في رحلة تعلم البرمجة مع هذه الدورة المميزة.

شرح الدورة

  • مقدمة إلى بايثون
  • التثبيت
  • المتغيرات والأنواع
  • الهياكل الشرطية
  • الحلقات
  • الدوال

ما هي بايثون؟: بايثون هي لغة برمجة عالية المستوى، سهلة القراءة، وديناميكية. تُستخدم في مجالات متعددة مثل تطوير الويب، علم البيانات، الذكاء الاصطناعي، وغيرها.

مميزات بايثون:

  • سهلة التعلم.
  • مكتبات وأطر عمل متعددة.
  • مجتمع دعم كبير.

قم بتحميل بايثون من الموقع الرسمي: python.org.

تأكد من إضافة بايثون إلى متغير البيئة (PATH) أثناء التثبيت.

المتغيرات تُستخدم لتخزين البيانات:

أنواع البيانات:

  • int: أعداد صحيحة.
  • float: أعداد عشرية.
  • str: نصوص.
  • bool: قيم منطقية (True/False).

تستخدم للتحكم في تدفق البرنامج:

age = 18
if age >= 18:
    print("مؤهل للتصويت")
else:
    print("غير مؤهل للتصويت")

حلقة for:

for i in range(5):
    print(i)

حلقة while:

count = 0
while count < 5:
    print(count)
    count += 1

تستخدم لتجميع الكود في وحدات قابلة لإعادة الاستخدام:

def greet(name):
    return f"مرحبا، {name}!"

print(greet("Ali"))
نهاية الدورة

بايثون هي لغة برمجة قوية ومرنة. يمكنك استخدام الأساسيات التي تعلمتها لبناء تطبيقات صغيرة أو حتى مشاريع كبيرة. مع الممارسة، ستصبح أكثر إلمامًا بمفاهيمها المتقدمة. إذا كان لديك أي أسئلة أو تحتاج إلى توضيحات، فلا تتردد في طرحها!
